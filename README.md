# transvision

## Dependencies

+ mmdetection3d==v1.1.0
+ mmcv==2.0.0
+ mmdet==3.0.0

```shell
pip install -U openmim
mim install mmengine
mim install 'mmcv>=2.0.0rc4'
mim install 'mmdet>=3.0.0'
mim install "mmdet3d>=1.1.0"
```
